HTML <button>과 다른 점

  • 다양하고 색감있는 디자인.
  • 로딩중임을 버튼안에 표시 가능합니다.

    어떤 하부 엘리먼트에 'loading' 클래스가 설정되어 있으면 기본적으로 숨겨져 보이지 않습니다. 그러나 <x-button> 클래스에 loading이 더해지면 보여지게 됩니다. 다시 <x-button> 클래스에서 loading를 지우면 숨겨지게 됩니다.

  • 버튼의 사이즈와 칼라를 웹사이트의 성격에 맞게 바꿀수 있습니다.

    "기본스타일 변경"을 클릭하셔서 새로운 스타일을 경험하세요.

기본스타일 Try it

    <x-button class="primary"> 프라이머리 </x-button>
    <x-button class="accent"> 엑센트 </x-button>
    <x-button disabled 비활성화 </x-button>

Loading

Any element with "loading" class is hidden in default. If the button class contains "loading" the loading sign become visible. While it is loading, users won't be able to click the button.

Click the below buttons to see loading sign.

디폴트 프라이머리 엑센트

Click here to remove all loading signs.

no-bg

Button with "no-bg" attribute won't have background color, switching to text color.

디폴트 프라이머리 엑센트 비활성화 
  <x-button class="no-bg"> 디폴트 </x-button>
  <x-button class="primary no-bg"> 프라이머리 </x-button>
  <x-button class="accent no-bg"> Accent </x-button>
  <x-button disabled class="no-bg"> 비활성화 </x-button>

no-shadow

Button with "no-shadow" class wont' have raised effect.

  <x-button disabled class="no-bg no-border no-shadow"> 비활성화 </x-button>

Icon

Button with "icon" class will have rounded shape instead of rectanble shape.
