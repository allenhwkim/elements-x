.app x-color-picker { }

Features

  • Fill value of input field when target is provided
  • Fires 'x-color-changed' event when color changed.

Usage with x-overlay / input

<input id="color" /> <x-overlay trigger="color" id="clock-picker"> <x-color-picker target="color"></x-color-picker> </x-overlay>

Usage with input

<input id="color" width="400" height="300"/> <x-color-picker target="color"></x-color-picker>

Attributes

  • width: width of color pallette. e.g., 240
  • height: height of color pallette. e.g, 180
  • trigger: id of input field

    • Events

  • x-color-changed: a custom event. selected color is in event.detail
