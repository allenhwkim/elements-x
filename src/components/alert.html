Alert

Features

  • close dialog when background clicked
  • when esc pressed close dialog
  • when open, trap focus within it.
  • when closed, focus back to trigger element
  • fire event when opened or closed
You can open a dialog with elementsX.showAlert(message, options) elementsX.showAlert('Alert Message',{ title: 'Alert Title', buttons: ['Cancel', 'Ok'] });

elementsX.showAlert() function returns a Promise that resolves text. You can use this to do something after the dialog is closed. e.g., `close`, `cancel`, `ok`

elementsX.showAlert('Alert Message', {buttons: ['Ok'] }) .then(resp => console.llg(resp));

To ensure user clicked one of buttons, pass notClosable as true. It will

  • Hide top-right close button
  • Not close alert box when background clicked
  • Not close when ESC key pressed
elementsX.showAlert('Alert Message', {buttons: ['Ok'], notClosable: true});

Options

NameDescription
title Optional, Title of message
message Required, Message of this alert, required
buttons Optional, Buttons on the bottom of this alert.
notClosable When true, default background click and top-right close button is hidden. Default. false
