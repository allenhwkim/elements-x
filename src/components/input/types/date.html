<x-input type="date" text-field=""></x-input> <x-input type="date" locale="ko-KR" value="2020-09-14" week-format="short" min-date="2020-08-14" max-date="2020-10-14"></x-input>

Attributes

  • locale: Default 'en-US'
  • date: Startig date. e.g. '2020-09-14'
  • week-format: Default 'short', 'long', 'short', or 'narrow'
  • min-date: Minimum calendar date selectable. default. '1969-01-01'
  • max-date: Maximum calendar date selectable. default. '2100-12-31'
  • first-day-of-week: Default 0, which is Sunday
  • date-format: Default 'yyyy-MM-dd', Format of date

    • Events

  • x-select: fired when date is selected with value in Date type
