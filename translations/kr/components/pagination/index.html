전체 레코드 수, 페이지당 레코드 수, 현재 페이지 번호에 따라 사용자가 특정 페이지를 선택할수 있는 기능을 제공합니다. The <x-pagination> element enables the user to select a specific page from a range of pages that are composed by total records, number of records per page, and the current page number.

<x-pagination value="4" num-records-per-page="10" num-pages="5" total-records="1000"> </x-pagination>

페이지가 선택되면 'x-select'를 발생합니다.

아래의 예제에서 다양한 옵션을 바꿈으로서 페이지 인덱스가 어떻게 바뀌는지 참조하세요.

Example Try it

10
5
1000
4

x-select 이벤트 내역 Empty

속성

  • current-page: 현재 페이지 번호
  • num-records-per-page: 전체 레코드의 수
  • num-pages: 보여줄 페이지의 갯수
  • total-records: 보여줄 페이지의 갯수

    • 이벤트

  • x-select: 페이지가 선택될때 발생하는 이벤트 (page, offset, limit를 포함합니다.)
