x-listeners

This is used to remove listeners automatically when this element is destroyed. It's useful when you add a listeners to a document-level or window-level.

To Test this page, click document. Then move to other page, then click the document again. Because a click listener is added to document, it's supposed to show 'doc licked' in console.

However, you won't see the message in console because the listener is removed by <x-listeners> when it is dosconnected.

// HTML <x-listeners debug></x-listeners> // Javascript function clickHandler(event) { console.log('doc clicked'); } document.addEventListener('click', clickHandler); document.querySelector('x-listeners').handleGlobalListener(document, 'click', clickHandler);
