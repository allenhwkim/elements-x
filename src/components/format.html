format() function

To use it with Javascript, do smartform.format(value, type, options)

  • type: 'currency', 'date', or 'number'
  • options:
    • locale: Optional, default 'en-US', e.g. 'en-US', 'ko-KR', 'en-FR', or 'en-IN'
    • maxDecimal: Optional, default 2, e.g, 3, 1, or 0
    • currency: Optional, default 'USD', e.g., 'KRW', 'INR'
    • format: Optional, default 'yyyy-MM-dd'
import { smartform } from '@smartform/smartform'; smartform.format(1234567.89, 'currency', {locale: 'fr-CA', currency: 'CAD'});

Attributes

Name Default Optional Description
type N/A Required format type. number, currency, or date
locale en-US Optional language-country. e.g. fr-CA
max-decimal 2 Optional max decimal point. Used with type="number"
currency USD Optional currency. e.g. USD, KRW, or CAD. Used with type="currency"
format locale date format. for en-US, mm-dd-yy Optional Refer to the Date Formats. Used with type="date"
