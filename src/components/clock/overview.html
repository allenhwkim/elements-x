Features

  • Shows an analog clock with give hour and minute.
  • With run="true", clock runs by itself
  • Updatable hour and minute by dragging hour hand or minute hand.
  • Fires `x-time-selected` when time changes.
  • Diaplay a clock on a page flow or as a dropdown

    It can be used with a button or input tag

Usage with input field

To update time, press tab to select hour hand or minute hand, Then, press arrow keys; up/down/left/right to update time. Or, simply, drag hour hand or minute hand to adjust time.

<input /> <x-overlay> <x-clock></x-clock> </x-overlay>
<x-clock run="true"></x-clock>

Attributes

  • hour: 0 to 24
  • minute: 0 to 59
  • target: id of element to be updated when a clock is updated.

    • Events

  • x-time-selected: a custom event. selected time is in event.detail
