Features

  • Provide navigation using keyboard
  • Fire 'x-select` evnet when a list item is selected
  • With 'selected' attribute, it highlights selected item
  • Accessibility features; aria-expanded / aria-has-popup

Default

  • File
  • Edit
  • Format
  • View
  • Help
Select an list item from the list.
<x-ul selected="file"> <ul> <li id="file"> File </li> <li> Edit </li> <li> Format </li> <li> View </li> <li> Help </li> </ul> </x-ul>
