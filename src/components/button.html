How it is different from html <button>

  • Better look-and-feel.
  • Showing loading sign.

    An element with 'loading' class is hidden in default. It shows up when x-button has 'loading' class.

  • Customizable color and button size.

    Try different styling by clicking "Overvide Default Style".

Default Try it

Default Primary Accent Disabled No Style 

    <x-button> Default </x-button>
    <x-button class="primary"> Primary </x-button>
    <x-button class="accent"> Accent </x-button>
    <x-button disabled Disabled </x-button>
    <x-button class="no-style"> No Style </x-button>

Icon

Button with "icon" class will have rounded shape instead of rectanble shape.

NameDescription
primary primary styling, background blue, color white
accent accent styling, background red, color white
loading It does not hide elelement with class with 'loading'
no-bg Shows transparent background color and changes color to black
no-border Shows transparent border
no-shadow Does not show shadow
