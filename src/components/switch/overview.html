Features

  • Modifiable On/Off Text
  • Customizable color and button size.

    Try different styling by clicking "Overvide Default Style".

Default

OFF ON OFF ON

<x-switch checked="checked"> <b>OFF</b> <b>ON</b> </x-switch> <x-switch> <b>OFF</b> <b>ON</b> </x-switch>

Custom

No Yes No Yes

<x-switch checked="true" color="blue" size="24px"> <b>No</b> <b>Yes</b> </x-switch> <x-switch color="blue" size="24px" checked="false"> <b>No</b> <b>Yes</b> </x-switch>

DISABLED:
OFF ON OFF ON

<x-switch checked="checked" disabled="disabled"> <b>OFF</b> <b>ON</b> </x-switch> <x-switch disabled="disabled"> <b>OFF</b> <b>ON</b> </x-switch>

Attributes

  • size: Default 32px, height of the switch button
  • color: Default red, color of the switch button
