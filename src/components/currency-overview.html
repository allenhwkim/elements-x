Features

  • Convert number to currency or any number format using Intl.NumberFormat

    • Attributes

    Reference: https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Intl/NumberFormat/NumberFormat
  • locale: Default en-US
  • style: Default currency
  • currency: Default USD
  • unit: The unit to use in unit formatting
  • unit-display: The unit formatting style to use in unit formatting
  • use-grouping: Whether to use grouping separators
  • notation: The formatting that should be displayed for the number
  • compact-display: Only used when notation is "compact". Takes either "short" (default) or "long".
  • currency-display: How to display the currency in currency formatting
  • currency-sign: accounting or standard
  • numbering-system: numbring system, e.g. arab
  • sign-display: When to display the sign for the number
  • minimum-integer-digits: The minimum number of integer digits to use.
  • minimum-fraction-digits: The minimum number of fraction digits to use.
  • minimum-fraction-digits: The maximum number of fraction digits to use
  • minimum-significant-digits: The minimum number of significant digits to use
  • maximum-significant-digits: The maximum number of significant digits to use.
