기능

  • 정렬기능
  • offset과 주어진 수에 따라 제한해서 보여주는 기능

예제

다음의 나라명 데이타는 https://restcountries.eu에서 찾을수 있습니다.

템플릿과 데이타만을 제공함으로서 검색과 정렬, 그리고 필터링이 가능한 테이블을 만들수 있습니다.

속성

  • order: 기본값 null, 정렬 표현식 예 "id desc"
  • offset: 기본값 0, 시작열 번호, 10일 경우 첫번째 보여지는 열은 11번째 열입니다.
  • limit: 기본값 0, 보여질 열의 수, 10일 경우 시작열번호부터 10개을 열을 보여줍니다.
  • list-from: <x-http> 엘리먼트 ID. 이것이 주어질 경우, 'x-success' 이벤트가 <x-http>에서 발생하면 그 데이타값으로 리스트를 변경합니다.
