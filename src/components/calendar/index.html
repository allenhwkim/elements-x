Features

  • Shows international language calendar

    English, Korean, Chinese, Arabic, Thai and more

  • Diaplay a calendar on a page flow or as a dropdown

    It can be used with a button or input tag

  • Highlights pre-selected date
  • Limit the calendar months using min. and max. date
  • Customizable color and dates display

    Try different styling by clicking "Overvide Default Style".
