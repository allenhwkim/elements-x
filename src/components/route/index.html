• Provides push-state implemention without writing Javascript.
  • When clicked, load different html to the given target.
  • When url changed, load different html from the url.
  • Fire 'x-route' event.
    • foo.html bar.html default.html
    This will be updated by route
    <x-route target="route-demo"> <a href="/component/route/foo" src="/components/route/foo.html">foo.html</a> <a href="/component/route/bar" src="/components/route/bar.html">bar.html</a> <a href="/component/route/default" src="/components/route/default.html">default.html</a> </x-route> <div id="route-demo"> This will be updated by route </div>

    Any element within <x-route> with "href" and "src" attribute sets route. "href" is the url to be when clicked, and "src" is the html to load when clicked.

    NameTypeDefaultOptionalDescription
    targetStringN/ARequired CSS selector expression of element that needs to be updated when route changes
    srcURLN/ARequired Child element attribute, indicates which url need to be downloaded when route is active
    hrefURLN/ARequired Child element attribute, indicates which url need to listen to trigger route change
    NameTypeDataDescription
    x-routeCustomEventRouteFired when route changes
