사용법

화면하단에 메세지를 잠시 보여주기 위해서는, `addSnackbar(message, options)` 함수를 샐행하세요. 예 addSnackbar('hello message')

옵션

  • class: 스낵바 디자인. 예, 'left', 'right', 'green' 혹은 'red'
  • duration: 스낵바가 보여줄 시간(밀리 세컨드). 예 1000

    • Example Try it


    <button class="left green">왼쪽에 녹색 스낵바</button> <button class="default">기본 스낵바</button> <button class="right red">오른쪽에 빨간 스낵바</button> <script> document.querySelectorAll('button').forEach(el => { el.addEventListener('click', _ => { const msg = el.className + ' ' + Math.ceil(Math.random() * 10**12); addSnackbar(msg, {class: el.className, duration: 1000}) }); }); </script>
