크기, 테두리, 배경색, 아이콘과 아이콘 색상을 변경할 수 있습니다.

체크박스 사용법 Try with static html

기본적인 체크박스입력대신에, 향상된 <x-input type="checkbox"> 를 사용해보세요

<x-input type="checkbox" fill="blue" icon="✖"></x-input>

<x-input type="checkbox" fill="red" icon="★"></x-input>

<x-input type="checkbox" radius="50%" icon="☻"></x-input>

<x-input radius="50%" icon="☻" type="checkbox"></x-input>

속성

  • size: 기본값 20px, 넓이와 높이
  • border: 기본값 #333, 테두리색
  • fill: 기본값 red, 배경색
  • icon: 기본값 '\2714', 아이콘
  • color: 기본값 inherit, 아이콘 색상
  • radius: 기본값 4px, 테두리의 둥근 정도
