Usage

call `addSnackbar(message, options)` function to display bottom appearing messages. e.g. addSnackbar('hello message')

options

  • class: look-and-feel of snackbar. e.g., 'left', 'right', 'green' and 'red'
  • duration: duration of message to be visisible in ms. e.g. 1000

    • Example Try with static html Try with Angular


    <button class="left green">Show Left green Snackbar</button> <button class="default">Show Default Snackbar</button> <button class="right red">Show Right red Snackbar</button> <script> document.querySelectorAll('button').forEach(el => { el.addEventListener('click', _ => { const msg = el.className + ' ' + Math.ceil(Math.random() * 10**12); addSnackbar(msg, {class: el.className, duration: 1000}) }); }); </script>
