Features

  • Showing sidebar with a blocker ONLY for a mobile device(less than 768px)
  • When the blocker clicked, it closes sidebsar
  • Disable page scrolling when sidebar is visible
  • Listening to swipe action. When left-swipe, it closes. When right-swipe, it opens
  • Listening to URL change. When changes, it closes sidebar
