Features

  • Provides custom look-and-feel of a html check box.

    size

    border

    background color

    icon

    icon color

Default

<label> <x-checkbox> <input type="checkbox" checked="checked" /> </x-checkbox> Checked </label>

Custom Checkbox with Attributes

<label> <x-checkbox fill="blue" icon="✖" size="24px"> <input checked="checked" type="checkbox"/> </x-checkbox> Check here </label>

Attributes

  • size: Default 20px, width and height of the checkbox
  • border: Default #333, border color of the checkbox
  • fill: Default red, background color of the checkbox
  • icon: Default '\2714', check icon
  • color: Default inherit, color of the check icon
