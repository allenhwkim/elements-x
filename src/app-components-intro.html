Collection of HTML elements eXtended.

elements-x is a collection of custom elements which is style-customizable and independent from any framework.

The goal of this library is to extend existing html elements; input, button, div, span, etc, for more functionality and custom styling.

By extending functionality of <div> and <ul>, it can even work as a minimal framework that can handle simple data binding and event handling based on the belief that most of DOM handling scripts can be wrapped into custom elements.

How to use

  • Add script to your page <script src="https://unpkg.com/elements-x"><script>
  • Write your html elements. <x-button class="primary">My Button</x-button>

Usage with npm

  • Install package $ npm install elements-x --save-dev
  • Import package import “elements-x”;
  • Write your html elements. <x-button class="primary">My Button</x-button>

How Is Different?

  • It’s custom-elements, which works as the same as html tags
  • NOT dependent to any framework; Angular, React, nor, VueJS
  • Customizable look-and-feel

    Unlike shadow DOM, every component has its own css in document level. This css can be overwritten by user's own css.
