Pagination

It enables the user to select a specific page from a range of pages that are composed by total records, number of records per page, and the current row number.

It fires 'select' event when a page is selected, so that a page-related setion. e.g., list, can react to it.

Default

<x-pagination onselect="console.log(event.detail)"></x-pagination>

Responsive To Attribute Change

It reacts to attribute value changes

Select Index
Select Number Per Page.
Select Number of Pages To Show.

Attributes

Name Description
index Defalt 0, Current index of records. Used to determine the current page.
num-per-page Default 10, number of records per page
num-pages Default 5, number of page buttons to display
total Default 100, number of total records

Events

Name Type Data Description
select Custom Event index, total, and numPerPage Fired when a page is selected
