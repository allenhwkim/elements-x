Open calendar with button

Selected: <button id="show-calendar">Show Calendar</button> <x-overlay id="calendar2" trigger="show-calendar" style="width: 280px"> <x-calendar></x-calendar> </x-overlay> Selected: <span id="date2"></span> <script> document.querySelector('#calendar2').addEventListener('x-calendar-selected', event => { document.querySelector('#date2').innerText = event.detail; document.querySelector('#calendar2').close(); // close overlay when selected }); </script>

Attributes

  • locale: default 'en-US'
  • date: startig date. e.g. '2020-09-14'
  • week-format: Default 'short', 'long', 'short', or 'narrow'
  • min-date: lowest calendar date selectable. default. '1969-01-01'
  • max-date: lowest calendar date selectable. default. '2100-12-31'
  • first-day-of-week: Default 0, which is Sunday

  • x-calendar-selected: fired when date is selected with value in Date type
