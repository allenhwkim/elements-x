기능

  • 주어진 시와 분에 따라 아날로그 시계를 보여줍니다.
  • run="true"가 명시되면 시계가 움직입니다.
  • 시침과 분침을 마우스로 움직여 시각을 변경할 수 있습니다.
  • 시각이 변경되면 `x-select`이벤트를 발생합니다.

Example Try with static html

시각을 변경하기 위해, 탭을 눌러 시침이나 분침을 선택하세요, 그리고, 화살표키를 눌러 시각을 변경하시면 됩니다. 혹은 마우스로 시침혹은 분침을 움직여 시각을 변경하시면 됩니다.

<x-clock run="true"></x-clock>

속성

  • hour: 0 - 24
  • minute: 0 - 59
  • target: 시각이 변경되었을 경우 업데이트할 엘리먼트 아이디.

    • 이벤트

  • x-select: 커스텀 이벤트. 선택된 시각은 event.detail에서 찾을수 있습니다
