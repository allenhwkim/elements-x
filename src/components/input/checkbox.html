Provides custom look-and-feel of input checkbox

with size, border, background color, icon, and icon color

Example Try with static html Try with Angular

Instead of a normal input element with checkbox type, Extend it, with <x-input type="checkbox">

<x-input type="checkbox" fill="blue" icon="✖"></x-input>

<x-input type="checkbox" fill="red" icon="★"></x-input>

<x-input type="checkbox" radius="50%" icon="☻"></x-input>

<x-input radius="50%" icon="☻" type="checkbox"></x-input>
