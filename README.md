## Use with npm install
```
$ npm install elements-x -D
```
Define all custom elements
```
import 'elements-x'; // Define all custom elements
```

## Use with browser
```
<head>
  <script src="https://unpkg.com/elements-x"></script>
</head>
<body>
  <x-clock run></x-clock>
</body>
```
