Features

  • Provides organized look-and-feel for tabs section and tab contents section.
  • The tab contents section change its contents by responding to selected tab.
  • With 'selected' attribute, a tab section is shown when initialized.
  • With 'disabled' attribute, it's unabled to see tab contents.
  • With 'ink-bar' class with tab index, it provides enhanced highlighting on aselected tab.
  • Customizable tab index section and contents section

    Try different styling by clicking "Overvide Default Style".
