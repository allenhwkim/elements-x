.app x-button { --primary: green; --accent: orange; --disabled: grey; --size: 32px; }

How it is different from html <button>

  • Better look-and-feel.
  • Disabled for a second when clicked.

    A button become automatically disabled for a second when clicked to prevent double click

  • Showing loading sign.

    An element with 'loading' class is hidden in default. It shows up when x-button has 'loading' class.

  • Customizable color and button size.

    Try different styling by clicking "Overvide Default Style".

Default

Default Primary Accent Disabled 

    <x-button> Default </x-button>
    <x-button class="primary"> Primary </x-button>
    <x-button class="accent"> Accent </x-button>
    <x-button disabled> Disabled </x-button>

Loading

Any element with "loading" class is hidden in default. If the button class contains "loading" the loading sign become visible. While it is loading, users won't be able to click the button.
