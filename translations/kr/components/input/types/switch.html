기능

  • 변경가능한 온/오프 문자
  • 색상과 크기를 변경할수 있습니다.

<x-input type="switch"></x-input>

다양한 사용법

<x-input type="switch" status="on" on="YES" off="No" color="blue"></x-input>

<x-input type="switch" status="on" on="ON" off="OFF" color="orange"></x-input>

<x-input type="switch" status="on" on="ON" off="OFF" disabled="disabled"></x-input>

속성

  • color: 기본값 red, 스위치 버튼 배경 색깔
  • on: 기본값 ' ', 상태가 온일 경우 문자
  • off: 기본값 ' ', 상태가 오프일 경우의 문자
  • status: 기본값 'off', 기본 상태, 'on' 혹은 'off'
  • disabled: 이 속성이 명시되면 비활성화됩니다.
