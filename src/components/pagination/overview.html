Pagination Element

The <x-pagination> element enables the user to select a specific page from a range of pages that are composed by total records, number of records per page, and the current page number.

<x-pagination value="4" num-records-per-page="10" num-pages="5" total-records="1000"> </x-pagination>

It fires 'x-page-selected' event when a page is selected, so that a page-related setion. e.g., list, can react to it.

Attributes

  • value: page number
  • num-records-per-page: number of records per page
  • num-pages: number of page buttons to display.
  • total-records: number of total records.

    • Events

  • x-page-selected: fired when a page is selected with value as page, offset, and limit
