Firebase Storage CRUD

How to upload, read, delete files with Firebase

Notes to upload and download from Firebase storage.
If you want to know about Firebase database, read ‘Firebase Basic CRUD’.

index.html

<script src="https://unpkg.com/firebase/firebase.js"></script>

main.js

firebase.initializeApp({
    apiKey: "[api_key]",
    authDomain: "[any_title].firebaseapp.com",
    databaseURL: "https://[any_title].firebaseio.com",
    storageBucket: "[any_title].appspot.com",
  });
var storageRef = firebase.storage().ref();
