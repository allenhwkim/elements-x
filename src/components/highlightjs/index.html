Feature

  • Convert a section to a syntax highlighted view, https://highlightjs.org/
  • Customizable theme
  • Prettify code with 'prettier' option
  • Fires 'load' event

Example

addEventListener('load', () => { const code = document.querySelector('#code'); const worker = new Worker('worker.js'); worker.onmessage = (event) => { code.innerHTML = event.data; }; worker.postMessage(code.textContent); }); <x-highlightjs theme="monokai" prettier onload="console.log('load', event.detail)"> addEventListener('load', () => { const code = document.querySelector('#code'); const worker = new Worker('worker.js'); worker.onmessage = (event) => { code.innerHTML = event.data; }; worker.postMessage(code.textContent); }); </x-highlightjs>
