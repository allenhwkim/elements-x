Message

Show a slowly disappearing message with elementsX.showMessage(message, options)

elementsX.showMessage('My Message' + new Date());

To control the time of the message disappearing, pass duration in options.

elementsX.showMessage('My Message' + new Date(), {type: 'success', duration: 1000});

To control type of the message, pass type in options as 'success' or 'error'.

elementsX.showMessage('My Message' + new Date(), {type: 'error', duration: 1000});

Options

NameDescription
duration Optional, Default 5000ms. After this period, message will disappear
