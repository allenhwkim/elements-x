Time selector for Input Field

<x-input type="time"></x-input> <x-input type="time" hour="10" minute="15" run="true"></x-input>

Attributes

  • hour: 0 to 24
  • minute: 0 to 59
  • rundefault false, if true, it runs the clock

    • Events

  • x-select: a custom event. selected time is in event.detail
