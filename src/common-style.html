CSS Classes

elements-x provides a set of css classes for your page.

  • heaing 1, 2, 3, and 4
  • shadow 1, 2, and 3
  • margin
  • border
  • padding
  • background
  • centered

Heading

use `x-h1`, `x-h2`, `x-h3` or `x-h4` class to show it as a header.

Overriding Default Settings

Set the following variables to override the default settings

body { --x-m: 1em; /* margin */ --x-b: 1px solid rgb(222, 226, 230); /* border */ --x-p: 1rem; /* padding */ --x-g: rgb(250, 250, 250); /* background */ }
