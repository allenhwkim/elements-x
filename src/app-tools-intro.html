Collection of useful tools for web development.

  • Unicodes: useful unicodes for html and css
  • SVG URL Encoding: format svg for url or css
  • URI Encoding: Encode URI for URL
  • Image To Base64: Convert image to base64 string for url or css
  • Format HTML: reformat html using prettier
  • Format CSS: reformat css or scss using prettier
  • Format JSON: reformat JSON using JSON.stringify function
  • Format Date: format date using Date.toLocaleDateStirng function
  • Format Number: format number using Intl.NumberFormat function
