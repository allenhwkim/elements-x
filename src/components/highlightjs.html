Feature

  • Convert a section to a syntax highlighted view, https://highlightjs.org/
  • Customizable theme
  • Prettify code with 'prettier' option
  • Fires 'load' event

Example Try with static html

When you use a framework like Angular and see no line breaks, please wrap innerHTML with <pre> because some framework does not keep the text format with its own compiler.

addEventListener('load', () => { const code = document.querySelector('#code'); const worker = new Worker('worker.js'); worker.onmessage = (event) => { code.innerHTML = event.data; }; worker.postMessage(code.textContent); });
