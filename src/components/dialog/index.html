Features

  • close dialog when background clicked
  • when esc pressed close dialog
  • when open, trap focus within it.
  • when closed, focus back to trigger element
  • fire event when opened or closed

    • Open Dialog

    You can open a dialog with a windows function "openDialog(message, options)"
  • message: paragraph of dialog
  • options
    • buttons: Buttons to be on the bottom of dialog. key is text of the button and value is a function to be performed when a button is clicked.
    • style: Custom style for dialog contents. e.g. {color: '#333'}
    • Open Dialog <script> function confirm() { openDialog('Do you want to continue', { buttons: { Yes: e => console.log('Yes button clicked', e), No: e => console.log('Nos button clicked', e) }, style: {border: '2px solid yellow', minWidth: '200px'} }); } </script> <x-button onclick="confirm()">Open Dialog</x-button>

    Attributes

  • closable: Default true, if false, only closable by close function()

    • Events

  • x-dialog-close: Fired when dialog close
  • x-dialog-open: Fired when dialog open
