속성들

  • locale: 기본값 'en-US'
  • date: 시작일. 예 '2020-09-14'
  • week-format: 기본값 'short', 'long', 'short', or 'narrow'
  • min-date: 선택가능한 최소 일자. 기본값. '1969-01-01'
  • max-date: 선택가능한 최대 일자. 기본값. '2100-12-31'
  • first-day-of-week: 기본값 0, which is Sunday
  • date-format: 기본값 'yyyy-MM-dd', 날짜 포맷 텍스트

    • 이벤트들

  • x-select: 날짜가 선택되었을때 이벤트가 발생합니다. 이벤트값의 타입은 Date입니다.
