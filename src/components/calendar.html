Calendar

Features

  • Multiple language expression with locale attribute. e.g., en, ko, or thai etc
  • Fire events when a date is selected and unselected
  • Updatable start day of week. Saturday, Sunday, or Monday
  • Holiday display using SDate.Holidays
  • Controllable date select with isSelectable(date) function
Selected Dates:
<x-calendar id="cal2" date="2022-04-01" multiple locale="ko" first-day-of-week="1"></x-calendar>

To show holidays on calendar, provide XCalanedar.HOLIDAYS as like the following.

import {XCalendar} from '@formhero/smartform-elements'; XCalendar.HOLIDAYS = [ {date: '2022-01-01', name: 'New Years\'s Day', Type: 'Statutory Holiday'}, {date: '2022-01-03', name: 'New Years\'s Day(in lieu)', Type: 'Government Holiday'}, {date: '2022-02-21', name: 'Family Day', Type: 'Regional Holiday'}, ... ];

To control selectable dates, provide XCalendar.IS_SELECTABLE(date) function. In default, all dates are selectable.

import {XCalendar} from '@formhero/smartform-elements'; XCalendar.IS_SELECTABLE = function(date) { const isHoliday = XCalendar.HOLIDAYS.find(h => h.date === date.toISOString().slice(0,10)); const isWeekend = date.getDay() === 0 || date.getDay() === 6; return !isHoliday && !isWeekend; }

Attributes

Name Default Optional Description
date Current Date Optional startig date. e.g. '2020-09-14'
first-day-of-week 0 Optional The start day of week
locale en-US Optional language/country
multiple N/A Optional If provided, It allows to select multiple dates
month-format long Optional format of month, default is 'long'". e.g., "numeric" (e.g., 1) "2-digit" (e.g., 01) "long" (e.g., January) "short" (e.g., Jan) "narrow" (e.g., J)
week-format short Optional format of week, default is 'short'". "long"(Monday), "short"(Mon), "narrow"(M)
