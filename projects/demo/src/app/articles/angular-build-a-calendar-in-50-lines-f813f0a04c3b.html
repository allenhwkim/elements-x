Angular: Build A Calendar In 50 lines

Minimal Calendar Widget Development

Building a calendar seems not easy and complicated. This is the instruction how to make it simple.

Steps to make it.

  • Get the first start day of calendar, not day ‚Äú1‚Äù, but the starting Sunday.
  • Gather all 42 days, 6 week days, starting from the starting Sunday
  • Then show these by 7 days a row
  • Extras; style it, add buttons, fire events

    • 1. Get the first start day of calendar, not day ‚Äú1‚Äù, but the starting Sunday.

    getCalendarStartDay(date = new Date) {
  const [year, month] = [date.getFullYear(), date.getMonth()];
  const firstDayOfMonth = new Date(year, month, 1).getTime();
  return this.range(1,7)
    .map(num => new Date(firstDayOfMonth - DAY_MS * num))
    .find(dt => dt.getDay() === 0)
}

    2. Gather all 42 days, 6 week days, starting from the starting Sunday

    getAllDays(date = new Date) {
  const calStartTime =  this.getCalendarStartDay(date).getTime();
  return this.range(0, 41)
    .map(num => new Date(calStartTime + DAY_MS * num));
}

    3. Then show these by 7 days a row

    <li class="days" *ngFor="let day of days"> {{day}} </li><li *ngFor="let date of getAllDays" (click)="selected.emit(date)">
  {{date | date:'d'}}
</li>
<style>
  ul { margin: 0; padding: 0; list-style: none;}
  li {
    display: inline-block;
    border: 1px solid #ccc;
    width: calc(100%/7 - 2px);
    text-align: center;
    cursor: pointer;
  }
</style>

    4. Extras; style it, add buttons, fire events

    When a date is clicked, fire selected event

    @Output() selected = new EventEmitter();

    When user want to see different months, provide a feature for it.

    setMonth(inc) {
  const [year, month] = [this.date.getFullYear(), this.date.getMonth()];
  this.date = new Date(year, month + inc, 1);
  this.dates = this.getCalendarDays(this.date);
}
