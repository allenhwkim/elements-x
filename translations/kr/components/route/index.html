x-route, URL변경시 페이지 콘텐츠도 변경

  • push-state를 이용하여 자바스크립트를 코딩하지 않고도 콘텐츠 변경
  • 클릭시 주어진 타겟에 URL에 맞는 HTML을 fetch하여 업데이트
  • URL이 변경시 URL에 맞는 HTML을 fetch하여 업데이트
  • route 변경시 'x-route'이벤트 발생

    • Example Try it

    foo.html bar.html default.html
    왼쪽의 링크를 클릭하면 이곳이 변경됩니다.
    <x-route target="route-demo"> <a href="/component/route/foo" src="/components/route/foo.html">foo.html</a> <a href="/component/route/bar" src="/components/route/bar.html">bar.html</a> <a href="/component/route/default" src="/components/route/default.html">default.html</a> </x-route> <div id="route-demo"> 왼쪽의 링크를 클릭하면 이곳이 변경됩니다. </div>

    route는 <x-route>엘리먼트의 하부 엘리먼트들의 'href'와 'src'에 따라 설정됩니다. 'href'는 클릭시 바뀔 URL이고, 'src'는 클릭시 바뀔 HTML의 위치입니다.
