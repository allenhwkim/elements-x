Git, How To Rollback Pushed Changes

This is how you rollback your recent commits to a certain point using git revert.

Let’s say you have the following log.

$ git log

commit **b1fcde85ef02de679d1246a6caf402d1f9290847**
0.7.0

commit **db6e674dbebba85d510d33de2d0fa24ecce55d2e**
upgraded with browser mode

commit **2ff779489b069e00bfb48aeb9c70716f9009fb2a**
added more commands and refactored commands; see, go to, and added src/web for browser mode

commit 3273827504fe1fb754e030786aa7bd5a2c6b141a
0.6.2

commit 543e63da81d1f13ce977a39313f33190f755a436
refactored with file names and variable names

commit 21ce08b576c4e6e6d20c769dc60cc1d3c7d4d400

You want to go back to 0.6.2, which is commit 3273875.

From the above, you want to revert the following commits;

b1fcde85ef02de679d1246a6caf402d1f9290847
db6e674dbebba85d510d33de2d0fa24ecce55d2e
2ff779489b069e00bfb48aeb9c70716f9009fb2a

Then, git revert these commits.

$ git revert b1fcde85ef02de679d1246a6caf402d1f9290847
$ git revert db6e674dbebba85d510d33de2d0fa24ecce55d2e
$ git revert 2ff779489b069e00bfb48aeb9c70716f9009fb2a

Now your git log will look like the following, which results in 3 commit messages.

$ git log**
commit 5d926a8fb66cd596c18aaa78e152d5182017a530
**Revert "added more commands and refactored commands; see, go to, amd addes src/web for browser mode"

This reverts commit 2ff779489b069e00bfb48aeb9c70716f9009fb2a.

**commit 04364fa9dd2150e25652364accb22efda5d36f18
**Revert "upgraded with browser mode"

This reverts commit db6e674dbebba85d510d33de2d0fa24ecce55d2e.

**commit e766d09dd871f303a3aba683b569e8daad092612
**Revert "0.7.0"

This reverts commit b1fcde85ef02de679d1246a6caf402d1f9290847.

commit b1fcde85ef02de679d1246a6caf402d1f9290847
0.7.0

However, you don’t like so many revert commits. You want a single commit for all 3 reverts.

To merge commit messages, use git rebase -i. -i means ‘interactive’.

$ git rebase -i
**pick** e766d09 Revert "0.7.0"
**pick** 04364fa Revert "upgraded with browser mode"
**pick** 5d926a8 Revert "added more commands and refactored commands; see, go to, amd addes src/web for browser mode"

To make a single commit message, change pick to squash except the first one.

**pick** e766d09 Revert "0.7.0"
**squash** 04364fa Revert "upgraded with browser mode"
**squash** 5d926a8 Revert "added more commands and refactored commands; see, go to, amd addes src/web for browser mode"

When you save this commit message, you will see one combined revert commit.

$ git log
**commit 48e2c0577ded94bc54ff4f3df677d5ddfeccb2bf**

Revert "0.7.0"

This reverts commit b1fcde85ef02de679d1246a6caf402d1f9290847.

Revert "upgraded with browser mode"

This reverts commit db6e674dbebba85d510d33de2d0fa24ecce55d2e.

Revert "added more commands and refactored commands; see, go to, amd addes src/web for browser mode"

This reverts commit 2ff779489b069e00bfb48aeb9c70716f9009fb2a.

commit b1fcde85ef02de679d1246a6caf402d1f9290847
0.7.0

commit db6e674dbebba85d510d33de2d0fa24ecce55d2e
upgraded with browser mode

commit 2ff779489b069e00bfb48aeb9c70716f9009fb2a
added more commands and refactored commands; see, go to, amd addes src/web for browser mode

commit 3273827504fe1fb754e030786aa7bd5a2c6b141a
0.6.2

That’s it. Now compare between current status and 0.6.2 version, 3273827.

$ git diff HEAD..`3273827
$ `

You will see nothing different. Rollback successful!!!
