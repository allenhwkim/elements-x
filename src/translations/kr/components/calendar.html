기능

  • 다양한 언어로 표현가능한 달력

    영어, 한국어, 중국어, 아랍어, 타이등등

  • 달력은 웹페이지 어디든 사용가능하고 혹은 드롭다운형태로도 가능합니다.

    버튼을 눌러 달력을 보여줄 수도 있고, 입력란에 클릭함으로써도 달력을 보여줄 수 있습니다.

  • 선택된 날짜를 강조해서 보여줍니다
  • 선택할수 있는 날짜값을 최소/최대값으로 제한할 수 있습니다.
  • 색상과 날짜 디자인을 바꿀수 있습니다.

Example Try with static html

버튼과 함께 사용하는 경우

달력을 보여주세요 선택된 날: <button>달력을 보여주세요</button> <x-overlay> <x-calendar target="date2"</x-calendar </x-overlay>

블럭과 함께 사용하는 경우짜



선택된 날짜: <div style="max-width:600px; border: 1px solid #ccc"> <x-calendar target="date1" locale="en-US"></x-calendar> </div> Selected: <span id="date1"></span>

속성

  • locale: 기본값 'en-US'
  • date: 시작일 date. 예 '2020-09-14'
  • week-format: 기본값 'short', 'long', 'short', 혹은 'narrow'
  • min-date: 선택가능한 최소일. 기본값. '1969-01-01'
  • max-date: 선택가능한 최대. 기본값. '2100-12-31'
  • first-day-of-week: 기본값 0, 일요일
  • target: 날짜가 선택되었을 경우 업데이트될 엘리먼트의 id

    • 이벤트

  • x-select: 날짜가 선택되었을 경우 Date 타입과 함게 이벤트를 발생합니다.
