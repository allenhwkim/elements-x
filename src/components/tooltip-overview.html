Feature

  • open tooltip when trigger element hovered, clicked, or focused
  • close tooltip when document clicked, mouseleave, or blur
  • when esc pressed, close tooltip
  • when open, trap focus within inside tooltip.
  • position tooltip underneath or over the trigger element
Show tooltip Hello Tooltip <span tabindex="0" id="show-tooltip">Show Tooltip</span> <x-tooltip trigger="show-tooltip"> Hello Tooltip </x-tooltip>

Attributes

  • trigger: id of element to activate this tooltip
