Feature

  • Convert a section to ACE Editor, https://ace.c9.io/
  • Customizable theme and language
  • Fires 'load', and 'change' events

Example Try it

function foo(items) { var x = "All this is syntax highlighted"; return x; } <x-ace onchange="console.log(event.detail)" onload="console.log(event.detail)"> function foo(items) { var x = "All this is syntax highlighted"; return x; } </x-ace>
