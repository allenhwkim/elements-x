기능

  • 키보드를 이용한 네비게이션 기능 제공
  • 리스트 아이템이 선택된 경우 'x-select'를 발생합니다.
  • With 'selected' attribute, it highlights selected item
  • 엑세스빌러티 기능 포함; aria-expanded / aria-has-popup

<x-ul selected="file"> <ul> <li id="file"> File </li> <li> Edit </li> <li> Format </li> <li> View </li> <li> Help </li> </ul> </x-ul>

리스트에서 하나의 아이템을 선택하시면 선택된 항목이 표시됩니다.

<x-ul id="my-list" class="dropdown" selected="redo"> ... 위의 항목과 동일합니다. ... </x-ul>

속성

  • selected: 선택된 리스트 아이템의 id, 기본값 null

    • 이벤트

  • x-select: 리스트가 선택되었을 경우 발생하는 이벤트
