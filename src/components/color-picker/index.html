.app x-color-picker { }

Features

  • Fill value of input field when target is provided
  • Fires 'x-select' event when color changed.

Usage with input

<input id="color" width="400" height="300"/> <x-color-picker target="color"></x-color-picker>

Attributes

  • width: width of color pallette. e.g., 240
  • height: height of color pallette. e.g, 180
  • trigger: id of element to be updated when a color is selected

    • Events

  • x-select: a custom event. selected color is in event.detail
