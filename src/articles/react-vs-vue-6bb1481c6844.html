React vs. Vue

This is comparison between ReactJS and VueJS.

As of 2017, VueJS is a rising star. At leaset from github code, it’s inevitable that it will become the number 1 starred repository in the world in 7 months based on 2017 trends;

React: 72.9 stars a day, 3.8 watchers a day
Vue: 111 stars a day, 5.8 watchers a day

So, I did little research to compare these two library(or a framework).

The following are minimal examples with Vue and React with an event, css, and html.

The biggest difference between React and Vue is the approach of coding. React treats everything as .jsx while Vue treat html as html and css as css.

As you see the minimal example with at least one event, little styling, and html, Vue is easier to understand and makes more sense than React.

If I were asked to choose one for a new project, I would personally choose Vue over React for this simple reason. “html is html, and css is css. They are not .jsx”
