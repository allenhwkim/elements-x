Simple Data Binding

Following examples demonstrate simple data binding and event handling using &x-div> and attributes.

This demo. shows the given name initially on both input field and message field. When input field is updated, the message field is also updated with the same value.

This example uses only one property, name, and processing flow for property 'name' is like the following.
  • One-time Property Inititialization
    When <x-div> starts, read 'name' attribute, then initialize "name" attribute.
  • One-way Data Binding
    When <x-div> property 'name' is assigned, it also binds value of elements having x-value attribute
  • One-way Event Processing
    When input event happens, it assigns input value to "name" property.
  • One-way Data Binding
    When <x-div> property 'name' is updated, it also binds value of elements having x-value attribute

<x-div name="Mr X"> Name: <input x-value="name" x-input="set(name)"/> <p>Hello <b x-value="name"></b> </p> </x-div>

Simple Style Binding

This demo. shows how to build a color using three input field, r, g, and b. This example uses three property, r, g, and b, and these are used to show a background color.

<x-div r="99" g="0" b="0"> R: <input type="range" max="255" value="99" x-change="set(r)"/> G: <input type="range" max="255" value="0" x-change="set(g)"/> B: <input type="range" max="255" value="0" x-change="set(b)"/> <div x-style="background:rgb({{r}},{{g}},{{b}})"></div> </x-div>
