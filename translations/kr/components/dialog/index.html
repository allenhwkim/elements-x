기능

  • 배경이 클릭되면 다이얼로그창이 닫힙니다.
  • ESC키가 눌러지면 다이얼로그창이 닫힙니다.
  • 다이얼로그 창이 열리면 포커스가 다이럴로그창안에서만 움직입니다.
  • 다이럴로그 창이 닫히면, 다이얼로그창을 연 엘리먼트로 포커스가 이동합니다.
  • 다이얼로그 창이 열리거나 닫히면, 이벤트를 발생합니다.

    • 다이얼로그 창 열기

    "openDialog(message, options)"함수를 사용해서 다리얼로그창을 엽니다.
  • message: 댜이얼로그 메세지
  • options
    • buttons: 다이얼로그 창 아랫쪽에 표시할 버튼들, key는 버튼의 문자덜이고 value는 버튼이 눌러졌을 경우 실행될 함수입니다.
    • style: 다이얼로그의 맞춤형 스타일. 예 {color: '#333'}
    • 다이얼로그 창 열기 <script> function confirm() { openDialog('Do you want to continue', { buttons: { Yes: e => console.log('Yes button clicked', e), No: e => console.log('Nos button clicked', e) }, style: {border: '2px solid yellow', minWidth: '200px'} }); } </script> <x-button onclick="confirm()"> 다이얼로그 창 열기 </x-button>

    Non-Closable Dialog

    닫을수 없는 다이얼로그 창 열기 이 다이얼로그는 배경회면을 클릭하거나 x 버튼을 눌러 닫을수 없슴니다.
    닫기     <button id="button2">닫을수 없는 다이얼로그 창 열기</button> <x-dialog trigger="button2" closable="false"> This is a non-closable dialog. <br/> <button onclick="this.closest('x-dialog').close()">닫기</button> </x-dialog>

    속성

  • closable:기본값 true, false일 경우 다이얼로그는 close() 함수만을 사용해서 닫을 수 있습니다

    • 이벤트

  • x-dialog-close: 다이얼로그를 닫을때 발생하는 이벤트
  • x-dialog-open: 다이얼로그를 열때 발생하는 이벤트
