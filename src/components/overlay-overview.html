Hello My Overlay

Feature

  • open overlay when trigger element focused or clicked
  • close overlay when document clicked
  • when esc pressed close overlay
  • when open, trap focus within it.
  • position overlay underneath or over the trigger element
  • when closed, focus back to trigger element
  • Customizable trigger actions
  • Customizable close actions
  • Customizable arrow display

    • Attributes

  • trigger: id of element to activate this overlay
  • show-arrow: Default false, for tooltip, show arrow
  • trigger-actions: Default click, actions to open overlay
  • close-actions: Default blur, actions to close overlay

    • Events

  • x-overlay-open: Fired when overlay opened with reason
  • x-overlay-close: Fired when overlay closed with reason
