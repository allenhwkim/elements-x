기능

  • 트리거 엘리먼트가 포커스가 오거나 클릭되면 오버레이가 열립니다..
  • Document가 클릭되면 오버레이가 닫힙니다.
  • ESC가 눌러지면 오버레이가 닫힙니다.
  • 오버레이가 열리면, 포커스가 오버레이안에서만 움직입니다.
  • 오버레이는 트리게 엘리먼트의 위 혹은 아래에 위치합니다.
  • 오버레이가 닫힐때 트리거 엘리먼트가 다시 포커스를 갖습니다.
  • 열림 이벤트, 닫힘 이벤트, 화살표 디자인을 맞춤화 할수 있습니다.

Example Try with static html

Show Overlay
<button>Show Overlay</button> <x-overlay> <div style="padding: 24px;"> <button>Hello</button> <button>Overlay</button> </div> </x-overlay>
Hello My Overlay
<x-input id="my-input" type="text" autocomplete="off"></x-input> <x-overlay trigger="my-input"> <div style="padding: 24px;"> Hello My Overlay </div> </x-overlay>

속성

  • trigger: 오버레이를 시작시킬 엘리먼트의 id
  • show-arrow: 기본값 false, 화살표를 보여줍니다.
  • trigger-actions: 기본값 click, 오버레이를 열어주는 이벤트들 예, 'focus, mouseenter, click'
  • close-actions: 기본값 blur, 오버레이를 닫는 이벤트들 예, 'blur, mouseleave'

    • 이벤트

  • x-overlay-open: 오버레이가 열릴때 발생하는 이벤트
  • x-overlay-close: 오버레이가 닫힐때 발생하는 이벤트
