Features

  • Shows international language calendar

    English, Korean, Chinese, Arabic, Thai and more

  • Diaplay a calendar on a page flow or as a dropdown

    It can be used with a button or input tag

  • Highlights pre-selected date
  • Limit the calendar months using min. and max. date
  • Customizable color and dates display

    Try different styling by clicking "Overvide Default Style".

Attributes

  • locale: default 'en-US'
  • date: startig date. e.g. '2020-09-14'
  • week-format: Default 'short', 'long', 'short', or 'narrow'
  • min-date: lowest calendar date selectable. default. '1969-01-01'
  • max-date: lowest calendar date selectable. default. '2100-12-31'
  • first-day-of-week: Default 0, which is Sunday

    • Events

  • x-calendar-selected: fired when date is selected with value in Date type
