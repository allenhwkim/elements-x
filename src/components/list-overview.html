Features

  • Convert ul/li list to hierarchy collapsable list
  • Provide navigation using keyboard
  • Fire 'x-list-selct` evnet when item is clickd or enter pressed
  • Select first item in default
  • When menu-style given as attribute, it responds to mouse hover
  • Accessibility full-feagtured
  • Highlight selected item

    • Attributes

  • menu-style: When given, the list works like menu by responding to mouseenter/mouseleave
  • select-first: Default true, select the first item when initialized
  • selected: the id of preselected menu item, default null

    • Events

  • x-list-select: Fired when menu item is selected by click or enter pressed
