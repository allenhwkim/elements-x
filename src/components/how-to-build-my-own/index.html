How To Build My Own Custom Element

You can build your own custom elements with customizable CSS and your own HTML by using built in common tools; setHTML, addCss, and removeCss.

  • setHTML: set HTML to the custom element. Returns a Promise
  • addCss: add scoped CSS to <head>
  • removeCss: remove scoped CSS from <head>

Example

import {setHTML, addCss, removeCss} from 'elements-x/common'; const css = ` x-button .x-button-container { display: flex; align-items: center; justify-content: center; }`; const html = `<div class="x-button-container"><slot></slot></div>`; class XButton extends HTMLElement { connectedCallback() { addCss(this, css); setHTML(this, html).then(_ => console.log('Yay!')); } disconnectedCallback() { removeCss(this); } } if (!customElements.get('x-button')) { customElements.define('x-button', XButton); }
