기능

  • 색깔을 선택하면 선택된 엘리먼트의 값이 변경됩니다.
  • 색깔을 선택하면 'x-select' 이벤트를 발생합니다.

입력란과 함께 사용하는 경우 Try it

<input id="color" width="400" height="300"/> <x-color-picker target="color"></x-color-picker>

속성

  • width: 칼라팔렛트 넓이. 예, 240
  • height: 칼라팔렛트 높이. 예, 180
  • trigger: 칼라가 선택되었을 경우 변경될 엘리먼트의 i

    • 이벤트

  • x-select: 맞춤형 이벤트, 칼라가 선택되었을 경우 선택된 색상을 event.detal에 포함해 이벤트를 발생합니다.
