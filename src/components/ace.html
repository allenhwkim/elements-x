ACE Editor

Features

  • Convert a section to ACE Editor, https://ace.c9.io/
  • Customizable theme and language
  • Fires Events, 'load','blur', 'change', 'copy', 'focus', and 'paste'

Default

<x-ace theme="chrome"> function foo(items) { var x = "All this is syntax highlighted"; return x; } </x-ace>
function foo(items) { var x = "All this is syntax highlighted"; return x; } function foo(items) { var x = "All this is syntax highlighted"; return x; }

With Gutter

<x-ace theme="chrome" gutter> function foo(items) { var x = "All this is syntax highlighted"; return x; } </x-ace>
function foo(items) { var x = "All this is syntax highlighted"; return x; } function foo(items) { var x = "All this is syntax highlighted"; return x; }

Different Modes

<x-ace theme="chrome" mode="text" gutter wrap-mode> Text Mode - Hello Ace Editor </x-ace>
Text Mode - Hello Ace Editor <h3>HTML Mode</h3> <div> <b>Hello Ace Editor</b> </div> // JavaScript Mode function foo(items) { var x = "All this is syntax highlighted"; return x; }

** Markdown Mode - Hello Ace Editor /* SCSS Mode */ .hello { .ace.editor { color: red; } } { "json": "mode", "hello": "Ace Editor" }

Attributes

Name Description
mode default javascript, https://cloud9-sdk.readme.io/docs/language-mode e.g. javascript, css, html, text, markdown, json, or scss
theme default monokai, Editor theme, https://cloud9-sdk.readme.io/docs/themes
gutterShow gutter
wrap-mode Text wrap mode

Events

Name Type Data Description
blur CustomEvent Fired when out of focus
change CustomEvent Object with detail Fired when content changes
copy CustomEvent Object with detail Fired when contents copied
focus CustomEvent Focus Event Fired when focused
load CustomEvent ACE editor instance Fired when starts
paste CustomEvent Object with detail Fired when contents pasted
