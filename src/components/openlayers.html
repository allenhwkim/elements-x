• Show a necessary map with simple HTML.
  • Show a marker with popup.
  • It does not require a access token.

Default Example static html Angular

<x-openlayers center="Hangang Bridge"></x-openlayers>

x-openlayers

Attributes

NameDescription
zoomDefault 15, Zoom level of the map
center Default 0, 0, Center Lng/Lat of the map, or an adderss. e.g. Incheon International Airport
satellite If given, satellite layer will be visible when initialized

x-ol-marker

Attributes

NameDescription
position Center Lng/Lat of the map, or an adderss. e.g. Incheon International Airport
center If given, this marker will be center of the map
