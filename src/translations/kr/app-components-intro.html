업그레이드된 HTML 엘리먼트 모음

elements-x는 custom elements들의 모음입니다. 스타일이 변경가능하고 프레임워크에 종속되지않고 별도로 편리하게 사용할수 있습니다.

elements-x의 목적은 기존에 존재하는 HTML태그들(input, button, div, span, etc)을 보다 다양한 스타일들과 많은 기능들로 확장하는 것입니다.

<div>태그 와 <ul>태그의 기능을 확장함으로서, elements-x가 미니 프레임워크로도 사용가능합니다. 대부분의 다이나믹 HTML은 DOM 모델 변경을 통해서 이루어지며, DOM 모델 변경은 간단한 데이타 바인딩과 이벤트처리방법을 태그 에트리뷰트를 통해 제공함으로서 가능합니다. 자세한 내용은 <x-div> 사용법을 참조하시기 바랍니다.

간단한 사용법

  • 기존의 웹페이지에 다은 script태그를 추가하세요. <script src="https://unpkg.com/elements-x"><script>
  • 이제 elements-x 태그들을 자유롭게 사용하세요 <x-button class="primary">My Button</x-button>

npm 사용법

  • elements-x 페키지릉 Install하세요. $ npm install elements-x --save-dev
  • elements-x 패키지를 import하세요. import “elements-x”;
  • 이제 elements-x 태그들을 자유롭게 사용하세요 <x-button class="primary">My Button</x-button>

다른 라이브러리와 프레임워크와 다른점

  • custom elements 기반입니다. 그래서 HTML 태그처럼 편하게 사용할수 있습니다.
  • 다른 프레임워크(Angular, React, VueJS, 기타등등)과 함께 사용할수 있지만, 거기에 종속되진 않습니다.
  • 스타일을 자유롭게 변경할 수 있습니다.

    shadow DOM은 유저가 스타일을 자유로이 변경할수 없지만, elements-x에서는 태그만을 위한 css를 사용합니다. 그리고, css는 언제든이 사용자가 다른 css를 사용하여 스타일을 자유로이 변경할수 있습니다.
